# URL Shortener (Kivy)

Простое приложение для сокращения ссылок через Pyshorteners(или через Cuttly в ранних версиях).

## Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Если вы используете старую версию:
4. **ВАЖНО:** Создайте в корневой папке файл `config.ini`.
5. Вставьте в него следующий текст, заменив `ВАШ_КЛЮЧ` на реальный API ключ от Cuttly (или API от других сервисов):

```ini
[cuttly]
API_KEY = ВАШ_КЛЮЧ
