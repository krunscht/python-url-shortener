# URL Shortener (Kivy)

Простое приложение для сокращения ссылок через API Cuttly.

## Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. **ВАЖНО:** Создайте в корневой папке файл `config.ini`.
4. Вставьте в него следующий текст, заменив `ВАШ_КЛЮЧ` на реальный API ключ от Cuttly (или API от других сервисов):

```ini
[cuttly]
API_KEY = ВАШ_КЛЮЧ
